# wordpress-rest-enum
A WordPress rest-enumeration script

# Install
- Pro Tip: utilize Python venv

`pip install -r requirements.txt`

# Usage
Enumerate users and media files: `python ./wordpress-rest-enum.py -w https://targetwebsite.com -u -m `
